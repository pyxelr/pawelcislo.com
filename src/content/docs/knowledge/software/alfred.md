My setup of [Alfred](https://www.alfredapp.com/):

- Activate with: `OPT` + `SHIFT` + `SPACE` (to allow for [Raycast](/knowledge/software/raycast/))
- Theme: `Alfred Modern Dark`
- Snippets for clipboard
- Workflows ([my GitHub list](https://github.com/stars/pyxelr/lists/alfred)):
  - [Alfred Almanac](https://alfred.app/workflows/giovanni/almanac) ← show weather with `!q` (keyword I've defined)
  - [Better Zip](https://macitbetter.com/library/betterzip/articles/betterzip-and-alfred/) ← add BetterZip actions into Alfred file menu
  - [Browser Tabs](https://github.com/epilande/alfred-browser-tabs) ← `chrome <tab>`
  - [DeepL Translation](https://alfred.app/workflows/alexanderwillner/deepl-translate/) ← `dlen` or `dlpl` (requires a [DeepL API key](https://www.deepl.com/account/summary))
  - [Default App](https://alfred.app/workflows/acidham/default-app/) ← `dapp` (requires `brew install duti`)
  - [DevUtils](https://devutils.com/) ← `dev <keyword>` (installed through [DevUtils](https://devutils.com/) settings)
  - [Open in iTerm2](https://github.com/iansoper/iterm2-alfred) ← add "Open in iTerm" into Alfred's action panel; however, a native option is under "Open with…"
  - [Open with VS Code](https://github.com/alexchantastic/alfred-open-with-vscode-workflow) ← `code` for search and `codeo` for opening current folder
  - [Power Thesaurus Search for Alfred](https://github.com/giovannicoppola/alfred-powerthesaurus) ← `pows`/`powa <word>`
  - [System Settings](https://github.com/alfredapp/system-settings-workflow/) ← open macOS system settings panes. Also, disable "Preferences" in "Default Results" not to duplicate results
  - [terminalfinder](https://github.com/LeEnno/alfred-terminalfinder) ← `ft/fi` / `tf/ti` (I've removed the Terminal and PathFinder parts)
  - [TimeZone](https://github.com/jaroslawhartman/TimeZones-Alfred) ← `!tz <time>` / `timezone`
  - [VSCodeDiff](https://alfred.app/workflows/logicxd/vscodediff/) ← `diff` (after selecting two texts)
  - [Visual Studio Code Project Manager for Alfred](https://github.com/kopiro/vscode-project-manager-for-alfred) ← `code <project>` (integration with [Project Manager](https://marketplace.visualstudio.com/items?itemName=alefragnani.project-manager) plugin)
  - [Window Switcher](https://alfred.app/workflows/alfredapp/window-switcher/) ← `[CMD]`+`[SHIFT]`+`[W]` or `!w` (without special character I will get irrelevant results)
- Extra setup
  - Disable _Show Spotlight search_ in macOS Keyboard options to use `[CMD]` + `[SPACE]` for Alfred
  - [Alfred Warp terminal features](https://github.com/mircolik/alfred-wasp-terminal-features) ← use Warp as a default terminal
  - [Custom Alfred iTerm Scripts](https://github.com/vitorgalvao/custom-alfred-iterm-scripts#copy-the-script) ← use iTerm2 as a default terminal
