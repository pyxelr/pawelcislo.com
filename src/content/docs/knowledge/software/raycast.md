My setup of [Raycast](https://www.raycast.com/):

- Raycast hotkey: `[CMD]` + `[SPACE]`
- Extensions
  - [Amphetamine](https://www.raycast.com/gstvds/amphetamine)
    - `amp` = Start New Session
    - `ampend` = End Current Session
  - [Brew](https://www.raycast.com/nhojb/brew)
  - [Browser Bookmarks](https://www.raycast.com/raycast/browser-bookmarks) (built-in)
    - `bb` = Search Browser Bookmarks
  - [Clean Keyboard](https://www.raycast.com/ike-gg/clean-keyboard)
  - [CleanShot X](https://www.raycast.com/Aayush9029/cleanshotx)
  - [Copy Path](https://www.raycast.com/koinzhang/copy-path) ← you can also hold `[OPT]` and use `[RMB]`
  - [DevUtils](https://www.raycast.com/devutils/devutils) ← you may just type e.g. `diff`
  - [Elgato Key Light](https://www.raycast.com/thomas/elgato-key-light) ← use `kon,koff` for "Toggle On/Off"
  - [Finder Utilities](https://www.raycast.com/edrf/finderutils) ← useful e.g. to `cf` (create file)
  - [Folder Search](https://www.raycast.com/GastroGeek/folder-search) ← "Search Files" also works but returns files as well
  - [GIF Search](https://www.raycast.com/josephschmitt/gif-search)
  - [Git Commands](https://www.raycast.com/manumorante/git-commands)
  - [Google Chrome](https://www.raycast.com/Codely/google-chrome)
    - `tt` = Search Tabs
    - `aa` = Search All
  - [Kill Process](https://www.raycast.com/rolandleth/kill-process)
  - [Microsoft Edge](https://www.raycast.com/KartikKumarSahoo/microsoft-edge)
    - `tt` = Search Tabs
  - [Port Manager](https://www.raycast.com/lucaschultz/port-manager) ← useful e.g. to search open ports from debugging and kill them. I prefer to disable "Open Ports in Menu Bar"
  - [Quick Calendar](https://www.raycast.com/fuksman/calendar)
  - [Quit Applications](https://www.raycast.com/mackopes/quit-applications)
  - [Recents](https://www.raycast.com/GastroGeek/recents) ← `rec` = Recents
  - [Remove Paywall](https://www.raycast.com/tegola/remove-paywall)
  - [Set Audio Device](https://www.raycast.com/benvp/audio-device)
  - [Shell History](https://www.raycast.com/koinzhang/shell-history)
  - [System Monitor](https://www.raycast.com/hossammourad/raycast-system-monitor)
  - [Terminal Finder](https://www.raycast.com/yedongze/terminalfinder) ← open settings, keep only the Warp-related options, and then type e.g. `fw` or `wf`
  - [Visual Studio Code](https://www.raycast.com/thomas/visual-studio-code)
    - `coder` = Search Recent Projects
  - [Visual Studio Code - Project Manager](https://www.raycast.com/MarkusLanger/vscode-project-manager)
    - `code` = main alias
  - [Warp](https://www.raycast.com/warpdotdev/warp)
  - [Word Count](https://www.raycast.com/itsmingjie/word-count) ← `words`
  - [Zed](https://www.raycast.com/ewgenius/zed-recent-projects)
