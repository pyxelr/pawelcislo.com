My setup of [Zed](https://zed.dev/):

- Theme:
  - Light: `Catppuccin Latte` (install within Zed)
  - Dark: [Dracula Solid](https://github.com/dracula-pro/zed)
- Settings in `~/.config/zed/settings.json` (accessible with _zed: open settings_)

  ```json
  {
    "theme": "Dracula Pro",
    "telemetry": {
      "diagnostics": true,
      "metrics": false
    },
    "ui_font_size": 15,
    "buffer_font_size": 15,
    "terminal": {
      "font_size": 13
    }
  }
  ```
