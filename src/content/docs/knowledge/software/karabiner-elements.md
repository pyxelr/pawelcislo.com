My setup of [Karabiner-Elements](https://karabiner-elements.pqrs.org/):

**Simple modifications** (used mostly for an external keyboard):

- `right_command` → `right_option` (to type Polish characters)
- `right_option` → `fn` (to quickly open the emoji menu)
- `fn` → `left_control` (for easier access to control)

**Virtual Keyboard**:

- Keyboard type: `ANSI` (with ISO, Alfred's snippet menu doesn't work)
