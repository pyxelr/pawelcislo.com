## Instructions

### Adding plugins

1. Install the plugin and its `.dll` or `.vst` should be added to one of the "Plugin search paths" (defined in FL Studio Plugin Manager)
2. _Add_ > _More plugins…_ > _Manage plugins_ > _Find more plugins_
3. If it's not defined, specify its _Type_ (_Effect_/_Synth_) by selecting the plugin
4. _Star_ the added plugin
5. Add it to the channel rack
6. Select the category in Plugin database
7. In plugin settings: _Add to plugin database (flag as favorite)_
   1. This will also add a thumbnail as per this [YT tutorial](https://www.youtube.com/watch?v=cBxMIqi-LN4&list=PLMzKVaLgphQcQVM1lrsi_rUWIBT68X-vG&index=21)
